Autism Spectrum Disorder (ASD) Support Website

   A web-based project developed to provide awareness, guidance, and supportive resources for children with Autism Spectrum Disorder (ASD) and their families.
This project focuses on improving understanding, early identification, and therapeutic support methods for autistic children.

Project Overview

  Autism Spectrum Disorder (ASD) is a neurodevelopmental condition that affects communication, social interaction, and behavior.

This website aims to:

 -Spread awareness about ASD
 
 -Provide early signs identification
 
 -Suggest therapeutic activities
 
 -Guide parents during meltdowns
 
 -Share treatment approaches

Highlight famous personalities with ASD

Features

 -Responsive homepage with navigation
 
 -Image slider using jQuery bxSlider
 
 -Informative sections about ASD
 
 -Early signs identification
 
 -Therapy and treatment information
 
 -Activities for autistic children
 
 -Educational, psychological, and pharmacological approaches
 
 -Information about well-known personalities with ASD
